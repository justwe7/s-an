# 食安

```js
├─src
│ ├─pages/            //小程序的页面
│ ├─components/       //组件目录
│ ├─utils/            //工具目录
│ ├─static/           //静态文件
│ ├─store/            //全局状态
│ ├─pages.json        //配置小程序页面
│ ├─manifest.json     //微信小程序的配置：appid、微信开发工具寻址的小程序目录等
│ ├─uni.scss          //uni-app提供的全局样式变量文件
│ ├─App.vue           //理解为小程序的app.js入口文件 包含小程序启动的生命周期
│ └─main.js           //挂载小程序页面 
├─project.config.json //微信小程序的配置：appid、微信开发工具寻址的小程序目录等
└─vue.config.js       //自定义的项目配置项
```
